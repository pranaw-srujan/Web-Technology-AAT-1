GALLERY

The gallery section showcases visuals related to web development, programming, and modern technology. These images represent the core concepts applied in this Web Technologies project.

The gallery uses Bootstrap’s grid system to ensure responsiveness across different devices such as mobile phones, tablets, and desktops. Each image automatically resizes according to screen size.

CSS hover effects are applied to the images using grayscale filters and scaling animations. When the user hovers over an image, it smoothly transitions from grayscale to full color, enhancing visual appeal and user interaction.

This section demonstrates the use of multimedia elements in web pages and highlights how images can be styled and animated using CSS to improve user experience while maintaining performance.
